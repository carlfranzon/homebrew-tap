# homebrew-tap

Homebrew formulae for [carlfranzon](https://github.com/carlfranzon) projects.

## Install

```bash
brew tap carlfranzon/tap
brew install abr
```

## Formulae

| Formula | Description |
|---|---|
| [abr](https://github.com/carlfranzon/agent-beads-runner) | Portable parallel AI agent launcher using git worktrees + beads |
